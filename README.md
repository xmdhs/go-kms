# go-kms

KMS 服务器/客户端模拟器，Go 语言实现。

## 用法

### 启动服务器

```bash
go-kms server -ip 0.0.0.0 -port 1688
```

### 运行客户端

```bash
go-kms client -ip 127.0.0.1 -port 1688
```

## 参数

### 服务器参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-ip` | 0.0.0.0 | 监听地址 |
| `-port` | 1688 | 监听端口 |
| `-epid` | 自动生成 | ePID |
| `-count` | 0 | 客户端数量（0=自动） |
| `-hwid` | 364F463A8863D35F | 硬件 ID |
| `-db` | - | KmsDataBase.xml 路径 |

### 客户端参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-ip` | 127.0.0.1 | 服务器地址 |
| `-port` | 1688 | 服务器端口 |
| `-mode` | Windows8.1 | 产品模式 |
| `-name` | 自动生成 | 机器名 |

## 构建

```bash
go build -o go-kms.exe
```
